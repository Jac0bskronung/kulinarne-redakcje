# Centrum Informacji — Digest

Codzienne podsumowania AI newsów i wyników piłkarskich.
Generowane automatycznie o **5:30** (poranny) i **18:30** (wieczorny) przez GitHub Actions.

## Format plików

`YYYY-MM-DD-morning.md` / `YYYY-MM-DD-evening.md`

## Źródła AI

- Anthropic Blog (Claude, oficjalne ogłoszenia)
- TLDR AI (codzienne podsumowanie branży)
- The Rundown AI
- HackerNews (tematy AI/LLM)
- Simon Willison's Blog

## Źródła piłkarskie

- La Liga (football-data.org)
- Liga Mistrzów (football-data.org)
